Graphing

Inputing functions

MENU -> GRAPH (on screen) -> (input function) -> F6

Zooming

Option 1. (from graph) F2 -> F5 (auto zoom) (or other keys for zoom in/out)

Option 2. (from graph) F3 -> (input Xmin, max, Ymin, max)

Trace

(from graph) F1 -> (arrows to move to points or type number for exact x)

Solving Functions

Roots

(from graph) F5 -> F1 (then use arrow button to move to other roots)

Y Intercepts

(from graph) F5 -> F4 (then use arrow button to move to other intercepts)

Find Y given X

(from graph) F5 -> F6 -> F1 -> (enter number) -> ENTER

Find X given Y

(from graph) F5 -> F6 -> F2 -> (enter number) -> ENTER

Intersections of functions

(from graph) -> EXIT -> (go to Y2 on function list and enter second function) F6 -> F5 -> F5 -> (then use arrow button to move to other points of intersection)

Table

MENU -> TABLE (on screen) -> (input function) -> F6

Table of values

(from table) EXIT -> F5 -> (input start, end, step) ENTER -> F6

Other

Radians

SHIFT -> MENU -> (navigate to Angle) F2

Absolute value

OPTN -> NUM (on screen - use Fn keys to select) -> ABS (on screen - use Fn keys to select)

Derivative

OPTN -> CALC (on screen - use Fn keys to select) -> d/dx (on screen - use Fn keys to select) -> (enter function) -> ENTER

Integral

(from graph) F5 -> F6 -> F3 -> (enter lower bound x) ENTER -> (enter upper bound x) ENTER
