Casio Tutorial - Functions

Roots

(from graph) F5 -> F1 (then use arrow button to move to other roots)

Y Intercepts

(from graph) F5 -> F4 (then use arrow button to move to other intercepts)

Find Y given X

(from graph) F5 -> F6 -> F1 -> (enter number) -> ENTER

Find X given Y

(from graph) F5 -> F6 -> F2 -> (enter number) -> ENTER

Intersections of functions

(from graph) -> EXIT -> (go to Y2 on function list and enter second function) F6 -> F5 -> F5 -> (then use arrow button to move to other points of intersection)

Table

MENU -> TABLE (on screen) -> (input function) -> F6

Table of values

(from table) EXIT -> F5 -> (input start, end, step) ENTER -> F6
