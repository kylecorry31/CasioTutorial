Casio Tutorial - Graphing

Inputing functions

MENU -> GRAPH (on screen) -> (input function) -> F6

Zooming

Option 1. (from graph) F2 -> F5 (auto zoom) (or other keys for zoom in/out)

Option 2. (from graph) F3 -> (input Xmin, max, Ymin, max)

Trace

(from graph) F1 -> (arrows to move to points or type number for exact x)
